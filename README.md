## üåü Key Features

* **Real-time Collaboration:** Secure, instant community chat powered by **Socket.io** for direct communication.
* **Dual-Role Ecosystem:** Specialized profiles for **Individual Innovators** and **Verified Companies**.
* **Skill-Based Discovery:** A transparent system to match talent with job and internship opportunities.
* **Identity Verification:** Robust OTP-based email verification to maintain a secure and trusted community.
* **Cloud-Optimized Media:** High-performance image and file handling via **Cloudinary**.
* **Scalable Architecture:** Built with **Flask** and **PostgreSQL (Aiven)** for a reliable, production-ready backend.

## üöÄ Getting Started

1. **Clone the repository:**
```bash
git clone https://github.com/ledger-phonix/TechNest.git

```

2. **Install dependencies:**
```bash
pip install -r requirements.txt

```


3. **Configure Environment Variables:**
Create a `.env` file in the root directory:
```env
FLASK_SECRET_KEY=your_secret_key
DB_URL=your_aiven_db_url
CLOUDINARY_URL=your_cloudinary_url
MAIL_USERNAME=your_email
MAIL_PASSWORD=your_app_password

```


4. **Run the Application:**
```bash
python app.py

```
